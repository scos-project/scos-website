Before You Start

Have these ready for a smooth install:

  • PC: Windows-capable gaming PC connected to your TV
  • Memory: 16 GB RAM
  • Storage: 1 TB SSD (OS + games)
  • Controllers: USB gamepads; USB hub required for multiple controllers
  • Network: Ethernet connection is required to complete installation — plug the cable in before you begin
  • Peripherals: Keyboard & mouse only for installation
  • Apps: Unified Remote (mobile) if you need to type credentials outside Steam
  • Important: Do not plug in any controllers during installation and initial graphics driver setup. Plug them in after the screen flicker finishes and you reboot.

Known Issue (Public Beta)

Unified Remote silent-install bug: during installation it may be detected but not actually installed.

  • SCOS does not expose the Windows desktop after installation, so you cannot manually install apps later.
  • Impact: You can play Steam games that don’t require entering credentials outside Steam. Avoid titles that force external launcher sign-ins (e.g., Rockstar Games Launcher, some EA titles) until the fix ships.

Install SCOS (Fresh OS Install)

SCOS installs as a new operating system powered by Windows (from an ISO). It does not modify your existing Windows.

⚠️ Important: Installing SCOS to a disk/partition can erase data. Back up everything important before you begin.

  1. Back up your data.
  2. Download the SCOS ISO (Public Beta).
  3. Create a bootable USB (≥ 16 GB) using Rufus or BalenaEtcher and select the SCOS ISO.
  4. Plug in Ethernet to the PC before you start (required to complete setup).
  5. Leave controllers unplugged for now.
  6. Boot from the USB drive (use your BIOS/UEFI boot menu key, e.g., F12/Del/Esc).
  7. Select the target drive/partition for SCOS and start installation (⚠️ this may erase/format that drive).
  8. Hands-off install: SCOS automatically installs Windows, required components/drivers, and Steam.
When installation finishes, the system will reboot into Steam’s first-run (OOBE) experience.

After Installation

GPU Driver Setup (first minutes)

  • Windows Update will automatically fetch and install your GPU drivers.
  • The screen may flicker during this process — this is normal.
  • When it completes, reboot once so Steam can detect the GPU properly.
  • Only after the flicker/reboot, plug in your USB controller(s).

Why wait to plug in controllers?

If a controller is connected too early, Windows/SCOS may detect it as an audio device and select it by default. Waiting until graphics install finishes prevents your sound from being routed to the controller.

If sound still goes to the controller

  1. Open Steam Big Picture menu → Settings → In-Game Voice.
  2. Under Microphone, select “Select Device” to open the Windows sound popup.
  3. Controller-mouse emulation:
    • Hold the PS (or Xbox) button and move the right stick to move the cursor.
    • RB = left click, LB = right click.
    • (Xbox mapping not fully tested, but expected to behave similarly.)
  4. In the Playback tab, choose your TV/AVR/speakers device (often High Definition Audio Controller or Realtek Audio) and set it as Default.
  5. Optionally switch to the Recording tab and set your preferred Microphone as Default.
  6. Apply/OK and return to Steam.
  7. Important: while moving/clicking with the cursor, keep holding the PS/Xbox button the whole time.

First-Run in Steam (OOBE)

  1. Select Language
  2. Select Timezone
  3. Sign in to Steam (credentials or Steam Guard)

Daily Use

  • System boots straight to Steam Big Picture.
  • USB controllers at launch; Bluetooth is planned for a future version.
  • Controller-first navigation; keyboard/mouse are not required for daily use.

Games Compatibility Note (Beta)

Until the Unified Remote issue is fixed, avoid titles that require external launcher sign-ins (e.g., Rockstar / some EA) unless you completed those sign-ins on a standard Windows desktop beforehand.
