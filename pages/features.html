Features

SCOS turns a Windows PC into a living-room console for Steam. It boots straight into Steam Big Picture Mode, uses a controller-first interface, and focuses on one thing: gaming.

SCOS is independent and not affiliated with Microsoft or Valve.

Core Pillars

Console Experience

Boots into Steam Big Picture Mode with a clean, couch-ready UI—no desktop or taskbar in daily use.

Native Windows Compatibility

Run your Steam library natively on Windows—no Proton/compat layers to tune.

Controller-First

Navigate everything with a gamepad after setup. Keyboard/mouse only needed during installation.

Bloat-Free

Only what’s needed to play and watch. Keep focus on games, not background apps.

Shell & Boot Flow

Shell Replacement

SCOS replaces explorer.exe with Steam Big Picture Mode after installation.

Direct-to-Steam

Every startup goes directly to Steam Home. Follow the on-screen instructions to start playing.

Smart Power

Sleep, wake, and shutdown designed for living-room flow.

Quick Menu (Planned)

Controller overlay for Wi-Fi, audio, power, and profile switching.

SCOS feels like a console but keeps native Windows drivers and performance under the hood.

Profiles (Planned)

Gaming

Performance-forward defaults for your library and launchers.

Streaming

Media-friendly tweaks for apps and video playback.

Kids Mode

Simplified home and parental controls.

Preinstalled Apps

  • Steam (Big Picture Mode) — living-room UI and game launcher
  • EA app — ready for EA titles that require it
  • Unified Remote Server — for occasional text/input without a keyboard

More to come in a future version. Other dependencies (runtimes, redistributables) can be auto-installed by Steam when you launch games.

Known Issue (Public Beta)

Unified Remote silent-install bug: during installation it may be detected but not actually installed.

  • SCOS doesn’t expose the Windows desktop after installation, so you cannot manually install apps later.
  • Impact: You can play Steam games that don’t require entering credentials outside Steam. Avoid titles that force external launcher sign-ins (e.g., Rockstar Games Launcher, some EA titles) until the fix ships.

Recommended prep (until fix): Until the Unified Remote issue is fixed, avoid titles that require external launcher sign-ins (e.g., Rockstar / some EA) unless you completed those sign-ins on a standard Windows desktop beforehand.

Controller Support

  • USB-only at launch (wired controllers)
  • Bluetooth pairing is planned for a future version
  • Works with Xbox / PlayStation / XInput-compatible gamepads over USB

Requirements (Summary)

  • PC: Windows-capable gaming PC for your TV
  • Memory: 16 GB RAM
  • Storage: 1 TB SSD (OS + games)
  • Network: Ethernet required to complete installation (plug in before you begin)
  • Controllers: USB gamepads; USB hub required for multiple controllers
  • Peripherals: Keyboard & mouse only for installation
  • Apps: Unified Remote (mobile) for entering credentials outside Steam when needed

Roadmap

  • Alpha: Shell handoff to Steam, controller-only flow, Unified Remote integration
  • Public Beta: wider testing, per-game presets, parental controls, network setup from the couch
  • 1.0: Theme variants, accessibility suite, first-run wizard, Bluetooth controller pairing, more preinstalled launchers
