Frequently Asked Questions

FAQ

Answers to common questions about installing and using SCOS.

General

Is SCOS a new operating system?

Yes. SCOS is installed as a fresh operating system powered by Windows, from an ISO. It is not a program that tweaks your existing Windows install.

What’s the goal of SCOS?

Deliver a console-style living-room experience with native Windows compatibility. SCOS boots straight into Steam Big Picture and hides desktop clutter.

Who is SCOS for?

Players who want a Windows-based console feel on a TV, with full compatibility for Steam (and other PC launchers documented in the Docs).

Installation

How do I install SCOS?

Create a bootable USB from the SCOS ISO, boot from it, and follow the installer. See the step-by-step guide on the Download page.

Will installation erase my data?

It can. Selecting a target disk/partition may erase/format that drive. Back up your data before you begin.

Do I need Ethernet?

Yes. An Ethernet connection is required to complete installation. Plug the cable in before you start.

Why do I see screen flicker after install?

Windows Update is installing your GPU drivers. This is normal. When it finishes, reboot once so Steam detects the driver correctly.

Controllers

When should I plug in controllers?

After GPU drivers finish installing and you reboot. Plugging in too early can make Windows select the controller as an audio device.

USB vs Bluetooth?

USB-only at launch (wired controllers). Bluetooth pairing is planned for a future version.

Which controllers work?

Xbox/PlayStation and other XInput-compatible gamepads (over USB). Use a USB hub for multiple controllers.

Audio picks the controller — how do I fix it?

  1. In Steam Big Picture: Menu → Settings → In-Game Voice.
  2. Under Microphone, choose “Select Device” to open the Windows sound popup.
  3. Controller cursor: Hold the PS/Xbox button and move the right stick to move the cursor. RB = left click, LB = right click. (Xbox mapping expected similar.)
  4. In the Playback tab, set your TV/AVR/speakers device (e.g., High Definition Audio Controller or Realtek Audio) as Default.
  5. In the Recording tab, set your preferred Microphone as Default (if needed).
While moving/clicking the cursor, keep holding the PS/Xbox button the whole time.

Networking

Can I use Wi-Fi?

Installation requires Ethernet. For living-room setups without Ethernet drop, many users bridge via a Wi-Fi repeater with an Ethernet port.

Do I need Unified Remote?

If a game/launcher requires credentials outside Steam, the Unified Remote mobile app is typically used to input text (see known beta limitation below).

Unified Remote & Launchers

Known issue in Public Beta

The Unified Remote installer may be detected but not actually installed during setup. Manual installation isn’t possible later because SCOS doesn’t expose the desktop.

What’s the impact?

You can play Steam games that don’t require outside-of-Steam sign-ins. Avoid titles that force launcher logins (e.g., Rockstar/EA) until the fix ships, or pre-sign-in on a standard Windows install beforehand.

SteamOS vs SCOS

SteamOS (Linux) often relies on Proton/Wine to run Windows games. It’s fantastic for many titles, but can struggle with external launchers and strict anti-cheat. SCOS is Windows-based, so games and launchers run natively without a translation layer, while keeping a console-simple flow.

See Documentation → SteamOS vs SCOS for a detailed comparison.

Troubleshooting

Black screen after exiting Steam

Exiting Steam Desktop Mode can leave a black screen. Restart and you’ll boot back into Big Picture.

GPU not recognized by Steam

Wait for Windows Update to install drivers (the screen may flicker). Then do a single reboot.

No audio on TV/AVR

If the controller was selected as the default audio device, fix it with the steps in Audio picks the controller.

Privacy

SCOS avoids telemetry where it’s safe to do so and ships bloat-free. The website’s contact form is embedded from a third-party provider; data is handled by that provider. See Docs → Privacy & Data for details.
