SteamConsoleOS (SCOS)

Welcome to SteamConsoleOS (SCOS) — a custom Windows-based gaming console OS that brings the Steam Big Picture experience to your living-room PC. SCOS is a new operating system powered by Windows (installed fresh from an image) — not a program that tweaks your existing Windows install. It streamlines Windows 10 into a console-friendly environment with controller support, kiosk-style lockdown, and pre-installed gaming essentials.

Overview

SCOS transforms a standard Windows 10 PC into a console-like device optimized for gaming with controllers.

  • Auto-login & Steam Big Picture at startup
  • Controller & network driver integration
  • Locked-down UI to prevent accidental desktop access
  • Pre-installed essentials (see Documentation for details)
  • Custom scripts to suppress unwanted shortcuts and dialogs

The Problem & Our Solution

The Problem

Many players without a Steam Deck build living-room PCs with SteamOS (Linux). That setup often relies on compatibility layers like Proton/Wine to run Windows games. While amazing for many titles, it can stumble with:

  • External launchers (EA App, Rockstar Games Launcher, Ubisoft Connect)
  • Strict anti-cheat/DRM that expect native Windows behavior
  • Login overlays/services that don’t play nicely through Proton/Wine

Result: friction with EA Sports FC, GTA/RDR2, and similar titles, or lots of tweaking.

The Solution: SCOS

SCOS is Windows-based, so games and launchers run natively — no Proton/Wine layer. You keep Windows’ broad driver support and compatibility, but the experience stays console-simple: straight into Steam Big Picture, with desktop clutter locked away.

Powered by Windows — without the bloat

  • Native compatibility for Steam and popular PC launchers
  • Bloat-free baseline — only what’s needed to play and stream
  • Telemetry minimized/disabled where safe for a lean, privacy-respecting setup
You get the breadth of Windows with the simplicity of a console.

Why SCOS?

You want Steam’s performance and compatibility—minus the clutter. SCOS is the middle ground: Windows where it counts, console where it feels. Not a workstation. Not for productivity. Pure play.

No Proton/Wine worries

Native Windows compatibility — no translation layer.

No desktop clutter

No desktop or taskbar; controller-first living-room UI.

Built for the TV

Readable typography, big targets, simple dialogs.

Bloat-free

Only what’s needed to play and watch.

What SCOS Does

  • Replaces explorer.exe with Steam Big Picture Mode after install.
  • Boots straight to Steam on every startup.
  • Controller-only navigation post-install (keyboard/mouse only for setup).
  • Unified Remote can be used for occasional text/input tasks (see Docs for current beta limitations).
It feels like a console; you keep native Windows drivers and performance under the hood.
